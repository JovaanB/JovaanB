## 🇫🇷 Ingénieur logiciel
----------------------------------

J’aime concevoir et développer des applications web modernes, mais surtout, qui apportent beaucoup de valeurs aux utilisateurs finaux.
J’utilise principalement **Next.js / ReactJS** pour créer des interfaces intuitives et performantes.

Ma passion pour l’informatique a commencé très jeune : je voulais tout comprendre de cet univers fascinant.

Le sport occupe une grande place dans ma vie.  
Je suis passionné par les sports de combat, et plus particulièrement par la **boxe française**, que je pratique depuis plus de 20 ans et dont je suis **champion du monde**.

Si je devais choisir ma stack préférée :
- **Framework** : React.js / React Native / Next.js / .NET
- **Base de données** : SQL / MongoDB
- **CSS** : Material UI / TailwindCSS 

Avec ces outils, on peut construire quasiment tout ce qu’on imagine.  
Next.js est un framework permettant de développer des applications React optimisées grâce au rendu côté serveur et à la génération statique.  

J’adore ce que je fais. N’hésite pas à consulter mon portfolio ou à me contacter !

### A passionate software engineer
----------------------------------

I love building and designing web applications.  
I mainly use **Next.js / ReactJS** to create smooth, high-quality user interfaces, but above all, which bring a lot of value to end users.

My interest in computers started very early — I was always curious and eager to learn more.  
I still remember my very first website: it looked terrible but had a clear purpose — to showcase my cousin, a **world champion in French boxing**.

Sports play a huge part in my life.  
I’m a big fan of combat sports, especially **French boxing**, which I’ve been practicing for over 20 years — and I’m proud to be a **world champion**.

If you ask me about my favorite stack:
- **Framework**: React.js / React Native / Next.js / .NET
- **Database**: SQL / MongoDB
- **CSS**: Material UI / TailwindCSS  

With these tools, you can build almost anything.  
Next.js is a framework enabling features like server-side rendering and static site generation for React applications.  

I love what I do. Feel free to check out my portfolio or reach out anytime!
